# TermuxGUI
A simple script launcher for Termux:X11 that simplifies your desktop environment setup on Android.

### Requirements
- [Termux:X11](https://github.com/termux/termux-x11)

### Installation
```bash
git clone https://github.com/Blziko/termuxgui
cd termuxgui
chmod +x autoinstall.sh termuxgui.sh
./autoinstall.sh
ln -sf $HOME/termuxgui/termuxgui.sh $PREFIX/bin/termuxgui
```

### Usage
```bash
termuxgui
```
